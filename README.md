# Trader Performance vs Market Sentiment Analysis

## Objective
This repository contains the Round-0 assignment for the Data Science Intern role at Primetrade.ai. The goal of this project is to analyze how market sentiment (Fear/Greed) relates to trader behavior and performance on the Hyperliquid platform, uncovering patterns to inform smarter trading strategies.

## Repository Contents
* `notebook.ipynb`: The core Jupyter Notebook containing data preparation, behavioral segmentation, visual analysis, and K-Means clustering.
* `README.md`: Project documentation and setup instructions.
